Timepiece Archive

Web Design

2025

Webpage as a digital representation of a physical collection of watches, either inherited or collected. Using an array of metadata relevant to each watch, an event listener tracks the hover states of each image and responds with the appropriate metadata. Further exploring the notion of time, the main graphic uses the display grid property to create a functioning clock.
